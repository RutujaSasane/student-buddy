# Student Buddy

An AI-powered Chrome extension that acts like a coding mentor on LeetCode, providing progressive hints instead of directly revealing solutions.

## 🚀 Project Status

🚧 Under Development

## 🎯 Goal

Help students become better problem solvers by guiding them with hints instead of giving away complete solutions.

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- Chrome Extension (Manifest V3)
- AI API (Coming Soon)
